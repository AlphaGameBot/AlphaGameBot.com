Powerful Features

User Leveling System

Boost server engagement with an XP system where members level up by sending messages and participating in discussions.

Global Points Leaderboard

Track competitive standings with a server-wide and global points system that shows the top members across all servers.

Creator that Listens

Enjoy a bot that evolves with your needs. The creator actively implements user-suggested features and responds to community feedback.

Popular Commands

/user stats @username - Show user statistics

/xkcd - Get an XKCD webcomic

/minecraft java [address] - Get information about a given Minecraft server!

/meme [subreddit] - Get a meme from a given subreddit!

/trivia - Get trivia about almost anything

/about - Get information about the bot (and command suggestions!)

Frequently Asked Questions

How do I add AlphaGameBot to my server?

Click the "Add to Discord" button on our website, log in to your Discord account, and select the server you want to add the bot to.

Is AlphaGameBot free?

Yes! AlphaGameBot is completely free and open-source! Check out our GitHub!

Why was AlphaGameBot created?

If you want the real reason, I was bored and wanted to create something new.

If you want the marketable1 reason, I noticed how many Discord bots required payment to be great2, and I wanted to make an feature rich, open-source Discord bot for anyone to use
